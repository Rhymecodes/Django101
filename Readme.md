## Create Project
- Create a virtual environment
    * python -m venv env 
    * source ./env/Scripts/activate
- Install Django within the virtual environment (pip intall Django)
- Create a project 'django-admin startproject projectname
cd to the project folder "projectlema"
run dir/ls. If you see the manage.py file, run 'python manage.py runserver'

## Create App
- Same dir as manage.py file
- create app 'python manage.py startapp firstapp'
- create views.
    * function based views
    * class based views
- create urls.py file in app folder
